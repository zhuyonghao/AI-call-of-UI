## 开始使用

### 安装

首先，克隆仓库并安装依赖：

```bash
git clone git@github.com:zhuyonghao/AI-call-of-UI.git

cd AI-call-of-UI

yarn install
```

### 运行应用程序

要在开发模式下运行应用程序，请执行：

```bash
yarn dev
```

这将在 http://localhost:3000 上启动应用程序。
